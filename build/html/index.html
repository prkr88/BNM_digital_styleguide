Baseline Grid


The nominal baseline ratio is 15:20. That is where the body copy is 15px, the leading is 20px.

This can be used as 10pt font-size and 14pt leading for print.

Use these mesurements to ensure consistancy in the copy and alignment of elements.

Rem sizing can be used based from a body copy size of 15px.
