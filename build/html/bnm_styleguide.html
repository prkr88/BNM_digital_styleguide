Using Our Fonts

To ensure continuity accross our branded assests, we have made our fonts avaliable as css imports for web, and downloadable for print and presentation use. Include the following snipped in your CSS or download the font package to use them in your project.

Headings

DOCK11 HEADER

LFT ETICA HEADING


The heading style is can be either DOCK11 or LFT Etica Display Heavy with all characters in uppercase and slight letter-spacing. The heading style can be treated with an orange to yellow gradient from left to right when created in photoshop and used as an image. The prefered treatment for web is a solid orange or grey colour.


The LFT Etica Display typeface is designed for mixed print and web use. It contains uppercase, lowercase, and special characters. LFT Etica Display Heavy is comercially licensed by Brand New Media for print use only. Any use of this font online is licensed though Adobe TypeKit and must be attributed to Adobe as per their EULA.



h1 {
	font-family: Dock11, LFT Etica Display Heavy, Arial Black, Sans-Serif;
	font-weight: 700;
	letter-spacing: 0.1rem;
	text-transform: uppercase;
}

Source Sans Pro Regular

Our body typeface is Source Sans Pro Regular, it is designed for mixed digital and print use and includes uppercase, lowercase, and special characters. We're using it here as a demonstration of how it should look as body copy. On web it should be used at 15px with a line-height of 1.4rem. For print this is 12pt with 17pt leading. The ratio of point size to leading/line-height is 1:1.4.


The recommended weight is 400 (regular), with strong elements in 600 (semi-bold). When used in print or in text larger than 15px the weight may drop to 300 (light). The typeface is freeware and licensed through Google fonts for commercial use. 



p {
	font-family: Source Sans Pro, Arial, Sans-Serif;
	font-weight: 400;
	letter-spacing: 0.01rem;
	line-height: 1.4rem;
}

Optimising for Web

A lot goes into perfect typography for web. To setup the body, declare the font-size to 15px, and line-height to 21px and enable font smoothing for webkit browsers. On retina devices we can drop the font-weight to 300.



body {
	margin: 0;
	padding: 0;
	font-family: Source Sans Pro, Arial, Sans-Serif;
	font-weight: 400;
	font-size: 15px;
	line-height: 21px;
}

html, html a {
    -webkit-font-smoothing: antialiased !important;
    text-shadow: 1px 1px 1px rgba(0,0,0,0.004);
}

@media (-webkit-min-device-pixel-ratio: 2), (min-resolution: 192dpi) { 
	p, a {
		font-weight: 300;
	}
}
