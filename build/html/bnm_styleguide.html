Headings

The font-family used is Dock11 or LFT Etica Display Heavy with all characters in uppercase and slight letter-spacing. The heading style can be treated with an orange to yellow gradient from left to right. The prefered treatment is a solid orange or grey colour.


Dock11

Dock11 is designed for print and should be used sparingly. It has no lowercase and limited special characters and should be reserved for large headings. The typface is freeware and is avaliable from dafont for personal and commercial use.


LFT Etica Display Heavy

For cases where headings are required for web, or need special characters, LFT Etica Display Heavy may be used as a substitute. The LFT Etica Display typeface is designed for mixed print and web use. It contains uppercase, lowercase, and special characters. LFT Etica Display Heavy is comercially licensed by Brand New Media for print use only. Any use of this font online is licensed though Adobe TypeKit and must be attributed to Adobe as per their EULA.


Arial Black

The system font chosen is Arial Black. This should be only be used as a contingency and must not be used in branded material.

Body Copy

Our body copy is presented in Source Sans Pro Regular. On web it should be used at 15px with a line-height of 1.4rem. For print this is 12pt with 17pt leading. The ratio of point size to leading/line-height is 1:1.4.


Source Sans Pro Regular

Source Sans Pro is the default font used. It is a full UTF-8 font for mixed digital and print use and includes uppercase, lowercase, and special characters. The recommended weight is 400 (regular), with strong elements in 600 (semi-bold). When used in print or in text larger than 15px the weight may drop to 300 (light). The typeface is freeware and licensed through Google fonts for commercial use.


Arial

The system font chosen is Arial. This should only be used as a contingency and must not be used in branded material.

Stylesheet

Use this stylesheet as a boilerplate to set up typographic styling for web.


	
	/* Define custom fonts and import Google fonts */
	@import url("http://fonts.googleapis.com/css?family=Source+Sans+Pro:300,400,600,700");
	@font-face {
		font-family: LFT Etica Display Heavy;
		font-weight: 700;
		src: url('http://creative.brandnewmedia.global/bnm/digital_hub/src/assets/fonts/LFTEticaDisplay-Heavy.woff');
		/* This url is yet to be confirmed */
	}

	/* Setup browser defaults */
	body {
		margin: 0;
		padding: 0;
		font-family: Source Sans Pro, Arial, Sans-Serif;
		font-weight: 400;
		font-size: 15px;
		line-height: 20px;
	}

	/* Set Heading style */
	h1 {
		color: #F47A28;
		font-family: LFT Etica Display Heavy, Arial Black, Sans-Serif;
		font-wight: 700;
		font-size: 3rem;
		line-height: 3rem;
		letter-spacing: 0.1rem;
		text-transform: uppercase;
	}

	/* Set paragraph style */
	p {
		color: #000000;
		font-size: 1rem;
		font-weight: 400;
		line-height: 1.4rem;
	}

	strong {
		font-weight: 600;
	}

	/* Default link style */
	a {
		color: #F47A28;
		font-weight: 600;
		text-decoration: none;
	}

	a:hover {
		border-bottom: 2px solid #F47A28;
	}

	a:visited {
		color: inherit;
	}

	/* Blockquote styling */
	blockquote {
		display: block;
		background-color: #eee;
		border-left: 10px solid #F47A28;
		margin:20px;
		padding: 15px;
	}

	li {
		margin-left: 1.2rem;
	}
